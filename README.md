# HealthTrack

A full-stack healthcare analytics platform built with React, Node.js, Express, and MongoDB Atlas.

## Features

- JWT Authentication
- Patient Management
- Medication Tracking
- Health Logs
- Medicine Compliance Analytics
- Dashboard Reporting
- MongoDB Aggregation Pipelines

## Tech Stack

Frontend:
- React
- Context API
- CSS

Backend:
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- JWT
